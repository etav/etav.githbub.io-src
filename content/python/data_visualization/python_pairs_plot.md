# Creating a Pairs Plot using Python
One of my favorite functions in R is the pairs plot which makes high-level scatter plots to capture relationships between multiple variables within a dataframe. To my knowledge, python does not have any built-in functions which accomplish this so I turned to [Seaborn](http://seaborn.pydata.org/index.html), the statistical visualization library built on matplotlib, to accomplish this.


```python
#import seaborn
import seaborn as sb
sb.set(font_scale=1.35, style="ticks") #set styling preferences
%matplotlib inline
```

Load the example iris dataset


```python
iris = sb.load_dataset("iris")
iris.head()
```

```python

```

Change the bar plot to lines graphs.


```python
plot = sb.pairplot(iris, hue='species', diag_kind='kde')
```


![png](/images/sb_pair_plot/pairs2.png)


Change the palette of the plot.


```python
plot = sb.pairplot(iris, hue='species', diag_kind='kde',palette='husl')
```
